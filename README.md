# Smile
- This web app recognizes smiles in real time.
- It records a timestamp each time you recognize your smile.

## Environment
- front
  - vue
  - element-ui
- backend
  - Python 3.7
  - Flask
  - SQLAlchemy
  - OpenCV
  - Pipenv

## How to start for development

- front

```
$ cd front
$ npm i
$ npm run build
```

- backend

```
$ cd backend
$ pipenv install
$ pipenv run start
```
