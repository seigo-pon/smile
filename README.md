# Smile
- Web site is real time face detection.

## Enviroment
- front
  - vue
  - element-ui
- backend
  - Python 3.7
  - Flask
  - SQLAlchemy
  - OpenCV
  - Pipenv

## How to start for development

- front

```
$ cd front
$ npm run build
```

- backend

```
$ cd backend
- pipenv run start
```

- `http://127.0.0.1:5000` accesses.
